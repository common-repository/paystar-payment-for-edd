درگاه پرداخت پی استار برای افزونه EDD - فروش محصولات دیجیتال - وردپرس
برای درخواست پذیرندگی و دریافت اطلاعات ترمینال از لینک زیر اقدام نمایید :
https://my.paystar.ir/

== Description ==

with this plugin, you can easily add paystar.ir gateway to edd checkout system
با این پلاگین میتونید درگاه پرداخت پی استار رو به سیستم تسویه حساب و پرداخت افزونه EDD 
اضافه کنید و محصولات دیجیتالی خودتون رو به فروش برسونید


== Installation ==
 
just enter paystar.ir terminal (API-Key)
راه اندازی و پیکربندی افزونه نیاز به تنظیمات خاصی نیست. فقط کافیه کد ترمینال پذیرندگی خودتون رو در تنظیمات افزونه وارد کنید
برای ورود به صفحه تنظیمات افزونه از منوی دانلود ها ، زیرمنوی پیکربندی ، به سربرگ درگاه های پرداخت رفته و تیک درگاه پی استار را فعال کنید
